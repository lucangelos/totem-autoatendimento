# 🎓 Totem de Autoatendimento Acadêmico

Sistema web desenvolvido como Trabalho de Conclusão de Curso (TCC), com o objetivo de automatizar serviços acadêmicos através de um totem de autoatendimento para alunos.

## 📌 Sobre o Projeto

O Totem de Autoatendimento Acadêmico foi desenvolvido para reduzir a necessidade de atendimento presencial em serviços escolares simples, oferecendo uma interface intuitiva e acessível.

O sistema permite acesso rápido a informações acadêmicas e funcionalidades importantes da instituição.

---

## 🚀 Funcionalidades

- 📅 Consulta ao calendário acadêmico;
- 📚 Visualização de cursos técnicos;
- 🔁 Informações sobre rematrícula;
- ❓ Página de dúvidas frequentes;
- 🎧 Área de ajuda e atendimento;
- 🏥 Envio de atestado médico com captura por câmera;
- 📩 Envio automático do atestado por e-mail.
